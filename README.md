# Goal Planner

AI-powered goal tracking app. Describe any goal in one sentence and get an instant step-by-step plan generated by AI. Then track your progress, manage tasks, and actually finish what you start.

**[Live Site →](https://www.aigoals.app)**

## Features

- **AI goal breakdown** — enter a goal, get a full task plan generated by Llama 3.1
- **Drag-and-drop task management** — reorder tasks, drag to complete
- **Built-in timer** — track time spent on each task
- **Deadlines** — set due dates with visual overdue warnings
- **Progress tracking** — per-goal and overall completion stats

## Tech Stack

**Frontend:** React 19, TypeScript, TailwindCSS, React Query, dnd-kit, React Router

**Backend:** AWS Lambda (Python), API Gateway, DynamoDB, Cognito (Google OAuth)

**AI:** Hugging Face Inference API (Llama 3.1)

**Hosting:** Vercel (frontend), AWS (backend)
